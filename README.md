# Implementation-of-a-deterministic-left-parser-for-LL-1-grammar
Лабораторная работа №5 «Реализация детерминированного левого анализатора для грамматики LL(1)» по дисциплине «Математическая логика и теория формальных языков»

Построить множества FIRST и FOLLOW для каждого нетерминала грамматики
и таблицу выбора (lookup table) и матрицу отношений предшествования. На их ос-
нове реализовать детерминированный левый анализатор для проверки принадлеж-
ности цепочки грамматике LL(1) (или правый анализатор LR(1), если это эффек-
тивнее). Реализовать генерацию цепочки. Назначить произвольные семантические
действия для заданных продукций (например, генерацию машинноориентированного
кода, движение робота или алгоритма выхода из лабиринта). Весь лог распознавания
выводится на экран.

Вариант 11:
S → aB
B → aBB | b

Задачи:
1. Построить множества FIRST и FOLLOW для каждого нетерминала грамма-
тики.
2. Построить таблицу выбора (lookup table) и матрицу отношений предшество-
вания.
3. На основе построенных множеств и таблиц реализовать детерминированный
левый анализатор для проверки принадлежности цепочки грамматике LL(1).
4. Реализовать генерацию цепочек, принадлежащих языку.
5. Назначить произвольные семантические действия для заданных продукций
(в работе реализована генерация машинно-ориентированного кода для стековой ма-
шины: инструкции PUSH 1, DUP, PUSH 2).
6. Обеспечить вывод подробного лога процесса распознавания (состояние стека,
текущий символ, применяемое правило и генерируемая инструкция
